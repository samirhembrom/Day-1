CSS Clamp

I would explain CSS clamp property in this section. The css clamp function works with three values. The way it is written is clamp(min,avg,max). How these values work is depending upon the width of the viewport. The min value will considered when the viewport is small, and avg or middle value will be considered when the viewport is similar to that of tablet, and the last value "max" will be computed when the viewport is large enough for a laptop or larger desktop sizes.
